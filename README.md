# crew-builder

Game-inspired AI agent team builder

This file is auto-generated by `electron/update-readme.cjs`.

## Features
- Agent management: name, role, system prompt, LLM config (CRUD)
- Team builder: 1 leader + up to 5 members, name/description
- Task execution with streaming output
- Phase indicator: planning -> executing -> synthesizing
- Leader synthesizes member outputs into a final answer
- LLM provider configuration and connection test
- Local SQLite storage for agents, teams, and tasks
- Pages: /agents (AgentsPage), /team (TeamPage), /run (RunPage), /settings (SettingsPage)

## Tech Stack
- Electron + Vite + React + TypeScript
- SQLite(better-sqlite3)
- Tailwind CSS
